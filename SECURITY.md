# Security Policy

## Supported Versions

Versions below 1.0 and 2.0 will not receive security changes.   
The 1.1 and 2.1 LTS versions will receive security updates until their lifetime expiration date.    

| Version | Supported          |
| ------- | ------------------ |
| 2.1.x   | :white_check_mark: |
| <= 2.0   | :x:                |
| 1.1.x   | :white_check_mark: |
| <= 1.0   | :x:                |
| 0.4 (currenty)   | :x:                |
| 0.6 (vnext)   | :x:                |
